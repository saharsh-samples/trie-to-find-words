# trie-to-find-words
React app that can be used to solve find words that are combinations of given letters puzzles. See [companion blog](http://saharsh.org/2019/06/18/trie-to-find-words/).

## Build

### NPM

```
npm install

npm run build
```

### Docker

```
docker build \
    -t trie-to-find-words:0.1 \
    .
```
